배경 및 목적
Tourmaline은 Markdown 문서를 캔버스 위의 카드로 시각화하는 도구입니다. Obsidian 플러그인에서 Chrome 확장으로 포팅하여 더 넓은 사용자가 활용할 수 있도록 합니다.
# 배경
문제 정의
기존 Markdown 편집기는 문서 구조를 선형으로만 표현합니다. 복잡한 문서일수록 전체 구조 파악이 어렵습니다.
현재 상태
Obsidian 플러그인으로 구현된 핵심 캔버스 경험이 존재합니다. Chrome 포팅을 통해 독립 확장으로 제공할 예정입니다.
기술 스택
TypeScript · Manifest V3 · IndexedDB · File System Access API
